# Décisions et roadmap — Session 13 juin 2026

## ACCESSIBILITÉ — Décisions prises

### LSF (Langue des Signes Française)
- Intégration Elix LSF (21 000 signes, gratuit)
- Barre de recherche LSF permanente en bas si profil = Malentendant(e)
- Clic sur n'importe quel mot → lien vers le signe Elix
- Mode créatif LSF : la personne clique sur des signes pour décrire ce qu'elle veut créer (ex: "plage" + "bleu" + "femme" → l'IA génère l'image)
- Sous-titres LSF sur vidéos : mots-clés signés en superposition (pas des phrases, juste les mots importants)
- Message à la communauté malentendante : honnête sur les limites actuelles, invitation à contribuer
- WebSourd (avatar LSF professionnel) → quand budget disponible

### Malvoyants / Déficients visuels
- Commande vocale complète si profil = Malvoyant(e) (voice_commands.js prêt)
- Toute navigation et validation par la voix : "Mode DA", "Carré", "Générer"...
- Description vocale complète de chaque page avant interaction
- Clavier Braille virtuel (6 points → texte automatique) → roadmap
- Export format BRF pour impression Braille → roadmap
- Compatibilité lecteurs d'écran NVDA/JAWS → roadmap

### Neurodivergents
- Profil neurologique dans Accessibilité : TDAH, TSA, Dys, Malvoyant(e), Malentendant(e), Sensibilité non diagnostiquée
- Distinction importante : TDAH/TSA = diagnostiqués. "Sensibilité non diagnostiquée" pour les autres.
